# 4. 多用户操作系统

* Linux 没有盘符的概念，只有一个根目录 / ,所有的文件都在它的下面。
* 由于Linux是多用户操作系统，在 根目录 `/` 下面有一个 `/home` 文件夹用来存放多用户的文件夹。每个用户可以在 `/home` 文件夹下新建自己的文件夹，然后再自己的文件夹中存放自己的文件。不能去其他的文件夹下操作。
