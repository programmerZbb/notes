# 测试分类

常见的开发流程里，都有测试人员，这种我们称为黑盒测试，测试人员不管内部实现机制，只看最外层的输入输出，不如你写一个加法的页面，会设计 N 个case，测试加法的正确性，这种代码里，我们称之为 E2E 测试。

更负责一些的我们称之为集成测试，就是集合多个测试过的单元一起测试

还有一种测试叫做白盒测试，我们针对一些内部机制的核心逻辑使用代码进行编写，我们称之为单元测试

这三都是我们前端开发人员进阶必备的技能。
